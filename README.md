ComEnergía – Plataforma Digital de Intercambio de Servicios entre Comunidades Energéticas

ComEnergía es una plataforma web diseñada para conectar comunidades energéticas y empresas del sector, facilitando el intercambio de servicios, conocimientos y recursos técnicos.
El proyecto nace como propuesta para fortalecer la cooperación entre actores energéticos en Colombia, reducir la dependencia externa y promover una transición energética justa, colaborativa y sostenible.

Prototipo funcional:
https://com-energia-final-eqn4.vercel.app

//////////////////////////////////////////

Características principales

Registro e inicio de sesión de usuarios y empresas.

Listado de servicios energéticos disponibles.

Solicitud de servicios mediante formularios interactivos.

Panel de usuario con solicitudes enviadas y empresas asociadas.

Sistema de calificación y promedio dinámico por empresa.

Interfaz moderna, responsiva y animada.
